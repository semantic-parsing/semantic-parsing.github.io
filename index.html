Semantic Parsing

Semantic Parsing is the task of converting natural language texts into formal machine-understandable logical-form representations. This website surveys the work in this area.

About This Site

This site is an experiment: a living literature review that allows you explore, search and navigate the literature in this area, by following a taxonomy based on the underlying design principles of each model.

Contributing

This research area is evolving so fast that a static review cannot keep up. But a website can! We hope to make this site a living document. Anyone can add a paper to this web site, essentially by creating one Markdown file. To contribute, open a pull request in GitHub, by following these instructions for contributing.
